# Selfit - ボディメイク記録アプリ

「鏡を見るのが楽しみになる毎日へ」
見た目重視のボディメイクをサポートするWebアプリケーションです。体重だけでなく、写真や体調、食事を記録することで、理想の自分への道のりを可視化します。

## 主な機能

*   **日々の記録**: 体重、体脂肪率、自撮り写真、コンディションタグ、一言メモを記録できます。
*   **ダッシュボード**: 現在のステータス、今日のミッション、グラフなどを一目で確認できます。
*   **グラフ分析**: 体重と体脂肪率の推移をChart.jsを用いたグラフで可視化（期間切り替え可能）。
*   **食事管理**: 食事の写真、メニュー、カロリーを記録し、振り返ることができます。
*   **ミッション & レベル**: 継続を促すミッション機能と、経験値によるレベルアップシステム。
*   **レスポンシブデザイン**: PCだけでなくスマートフォンでの利用に最適化されています。

## 技術スタック

*   **Backend**: Python 3, Django
*   **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
*   **Database**: SQLite (開発環境)
*   **Libraries**: Chart.js (グラフ描画)

## ローカルでの実行方法

1.  リポジトリをクローンします。
2.  仮想環境を作成し、有効化します。
3.  依存ライブラリをインストールします。
    ```bash
    pip install -r requirements.txt
    ```
4.  データベースのマイグレーションを実行します。
    ```bash
    python manage.py migrate
    ```
5.  開発サーバーを起動します。
    ```bash
    python manage.py runserver
    ```
